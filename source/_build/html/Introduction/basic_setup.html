Basic Setup¶

首先，你需要从Unity Store上下载这个 Plugin.

然后，导入这个插件，你会发现在 Scene Folder 找到各种各样的Demo.

你可以选择打开任意一个Demo例如 Demo_File.scene ，然后启用 Renderer,就可以在场景中看到插件的效果。

你也可以继续看这篇文章，从零开始了解插件的使用过程。
